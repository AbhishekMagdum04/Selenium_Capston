# Project Tech Stack

## Selenium Capstone - Ecommerce Site Testing

### Testing Framework
- **Selenium WebDriver** (v4.19.1) - Web automation and browser testing framework
- **TestNG** - Test management and execution framework
- **JUnit** (v4.11) - Unit testing framework

### Build & Dependency Management
- **Maven** - Build automation and dependency management tool
- **Java** - Primary programming language

### Data Processing
- **Apache POI** (v5.2.3) - Excel file reading/writing for test data management
  - `poi` - Core library for MS Office documents
  - `poi-ooxml` - Extended support for Office Open XML formats

### Project Structure
- **Eclipse IDE** - Development environment (indicated by `.classpath` and `.project` files)
- **TestNG Configuration** - `testng.xml` for test suite configuration
- **Maven POM** - `pom.xml` for project configuration and dependencies

### Key Features
- Selenium 4.19.1 for modern web automation
- TestNG for advanced test management capabilities
- Apache POI for data-driven testing with Excel sheets
- Maven for consistent builds and dependency management

### Testing Approach
This is an **SDET (Software Development Engineer in Test)** project designed for:
- E-commerce website automation testing
- Data-driven test scenarios using Excel files
- Comprehensive test suite execution via TestNG
